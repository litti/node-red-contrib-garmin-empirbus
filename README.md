### Garmin Empirbus Connector
With this module you can connect to your Garmin Empirbus MCU or Serv7 display acting as an MCU.

I use to connect to my Eura Mobil Xtura Garmin Serv7 display but it may work in other motorhomes or in boats which use this equipment as well.
